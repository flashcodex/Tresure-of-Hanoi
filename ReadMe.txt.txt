To play the Treasure of Hanoi please Follow the following Instruction

-Please Save the whole file to the Drive C
-Please Install jdk-12.0.1

-if your Jdk bin Path is default no need to edit 
-Edit the Path(Location of the jdk bin) in the Open.cmd and InstallGame.cmd right click then open with notepad(any text editor) to edit

-Open the InstallGame.cmd(One Time only) 
	Note ! Font Installation Pop Up will show 4 times (Click the install button after installing close it and another Pop Up will show)
	     ! Do not close the cmd it will automatically close after installtion

-Open the Open.cmd
	Note ! The do not close the cmd it will automatically close it self when to exit the game
